## ⚡ The Vision: From Tenants to Homeowners

**Neuralis** is a global, self-healing **DePIN** (Decentralized Physical Infrastructure Network). We are stripping the "Intelligence Age" away from centralized mega-corporations and handing it back to the individuals who provide the power.

By turning idle silicon into a global supercomputer, Neuralis provides **free, private, and uncensorable AI** for every human on Earth.

> [!IMPORTANT]
> **The Neuralis Flip:** Instead of paying $20/month to rent a "corporate brain," you contribute your idle GPU/CPU cycles to the mesh and earn **sovereign intelligence** in return.

## 🛠️ Tech Stack (The Engine)

Neuralis is built on a high-performance, modular architecture designed for massive scale:

* **Networking:** Custom P2P Gossip Protocol (Layer 1)

* **Storage:** Sharded Erasure Coding (Reed-Solomon implementation)

* **Intelligence:** 4-bit Quantized Local LLM Orchestration

* **Security:** AES-GCM Encryption + Zero-Knowledge Proof (ZKP) Layer

* **Environment:** Reproducible builds via **NixOS** & `nix-shell`

Your node will begin discovering peers and syncing the global ledger immediately.
